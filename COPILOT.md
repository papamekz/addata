# GitHub Copilot Instructions

Use `AGENTS.md` as the general project guide and `SKILL.md` as the analysis
protocol for OOH/DOOH-related tasks.

Preferred data entrypoints:

- `data/digest.json` for compact English claim retrieval
- `data/index.json` for full machine-readable metadata
- `data/rag-chunks.jsonl` for retrieval-augmented generation
- `QUICKREF.md` for task-oriented navigation
- `DATA_QUALITY.md` for limitations and evidence handling

Do not treat quote/context records as empirical evidence.
